Alert
=====

A custom admonition with short text and a link to learn more.

Using:

.. code-block:: rst

   .. alert::
      :link: https://scylladb.com
      :link_text: Learn more about the change
      :icon: logs

      We’re updating our license & versioning policy.
    
Renders as:

.. alert::
   :link: https://scylladb.com
   :link_text: Learn more about the change
   :icon: logs

   We’re updating our license & versioning policy.
